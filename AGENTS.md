# AGENTS.md

## 产品与阶段范围
- 产品名称：creator-os。
- 产品模块：
  1. 账户体检（account-audit）
  2. 赛道地图（niche-map）
  3. 改稿引擎（rewrite-engine）
- 当前阶段仅开发「账户体检」完整逻辑。
- 「赛道地图」「改稿引擎」仅保留首页卡片与功能占位，不实现完整业务流程。

## 产品与设计约束
- UI 风格：黑底、高级、冷静、结构化、偏策略中台；避免通用 SaaS 视觉语言。
- 前端文案与交互：中文优先。

## 安全与平台边界
- 不做爬虫登录。
- 不做代替用户登录平台后台。
- 不做自动发视频。
- 抖音接入仅保留「官方 OAuth 授权」架构占位。
- 涉及真实抖音 OAuth 的实现位置需明确 `TODO` 标记。
